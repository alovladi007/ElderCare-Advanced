üÜò Troubleshooting

If you see "Port 3000 is already in use":

  • Press Y when React asks to use another port, OR
  • Kill the process: lsof -ti:3000 | xargs kill -9

If you get module errors:

rm -rf node_modules client/node_modules
npm run install-all
